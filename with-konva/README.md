# Summary
Next.js + TypeScript + Canvas([konva](https://github.com/konvajs/react-konva))

# Env
- next v11 (base webpack5)

# Tutorial

## Machine Local

```tsx
git clone git@github.com:ogasawaraShinnosuke/nextjs-examples.git
cd nextjs-examples/konva
yarn install
yarn dev
```

## Docker

```tsx
docker-compose build
docker-compose up
```
